Install from Source
-------------------

To install tenzing from source, clone the repository from `github
<https://github.com/ieaves/tenzing>`_::

    git clone https://github.com/ieaves/tenzing.git
    cd tenzing
    python setup.py install .


Testing
-------

Test tenzing with ``pytest``::

    cd tenzing
    pytest
