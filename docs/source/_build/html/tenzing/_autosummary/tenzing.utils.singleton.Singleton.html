tenzing.utils.singleton.Singleton¶

class tenzing.utils.singleton.Singleton¶

Singleton metaclass

Any instantiation of a Singleton class yields the exact same object, e.g.

>>> class MyClass(metaclass=Singleton):
>>>     pass
>>> a = MyClass()
>>> b = MyClass()
>>> a is b
True
